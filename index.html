T

Todo List

App · Tauri · Web

설명

할 일을 관리하는 앱입니다. 추가, 수정, 삭제, 완료 체크 기능과 카테고리/우선순위 분류, 필터링을 지원합니다. 웹 버전과 Android 앱으로 사용 가능합니다.

앱 흐름

할 일 입력 → 카테고리(일반/업무/개인/쇼핑/학습)와 우선순위(낮음/보통/높음) 선택 → 추가 → 체크박스로 완료 토글 → 수정 버튼으로 내용/카테고리/우선순위 변경 → 필터로 카테고리, 우선순위, 상태별 조회

사용 기술

Vanilla HTML/CSS/JS Tauri v2 + Rust LocalStorage 반응형 + 다크모드 GitHub Pages

주요 기능

  • 할 일 CRUD — 추가, 모달 수정, 즉시 삭제
  • 완료 체크 — 체크박스 토글 시 취소선 표시
  • 카테고리 분류 — 일반/업무/개인/쇼핑/학습 태그
  • 우선순위 설정 — 낮음(파랑)/보통(노랑)/높음(빨강) 색상 태그로 시각 구분
  • 필터링 — 카테고리, 우선순위, 상태 조합 필터
  • 통계 — 전체/완료/진행중 개수 실시간 표시

아키텍처

  • 단일 페이지 앱 — 페이지 이동 없이 DOM 조작으로 화면 전환
  • 단방향 데이터 흐름 — 상태(todos 배열) 변경 시 renderTodos()로 전체 재구성
  • LocalStorage 동기화 — 모든 조작 후 JSON으로 영구 저장
  • Tauri 연동 — 웹뷰가 Rust 백엔드 위에서 동작, 순수 웹 코드 그대로 사용
웹으로 방문 앱 다운로드 (APK)
